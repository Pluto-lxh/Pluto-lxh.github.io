文章中插入图片

发表于 2020-01-25 | 分类于 博客搭建

hexo引入本地图片

1、把站点配置文件_config.yml里的post_asset_folder这个选项设置为true。设置完这个参数之后，每次new命令创建文章的时候就会生成同名的资源文件夹，部署的时候就会把资源文件同步上传到文章目录下。

2、使用官方提供的标签

假设我们要插入一张1.jpg的图片，就先将图片复制到同名目录下，然后在需要插入图片的地方插入标签

图片就能完美的显示啦。

官方给出的建议是使用标签插入而非markdown方式，使用markdown方式会导致在首页或者归档页上无法正常显示图片。我试了一下在首页上图片确实没有显示出来。

遇到的一些坑

刚开始在网上找资料的时候，大多数教程都写的使用hexo-asset-image这个插件。我安装之后根本就用不了呀，想到有可能是路径问题或者是版本兼容问题。我按照这个方向去查找问题，路径试了个遍发现图片在markdown上能正常显示，在网页上就是显示不出来。于是百度查了一下，这才了解到在官网里显示最新版本的hexo已经加入了图片标签的支持。

总结：有问题还是多去看看官网呀！！！折腾了老半天，心累o(╥﹏╥)o

官网链接：https://hexo.io/zh-cn/docs/asset-folders

hexo搭建个人博客

发表于 2020-01-24 | 分类于 博客搭建

前言

正所谓工欲善其事必先利其器，在开始正式搭建博客前我们需要给我们的电脑部署好坏境，以便于我们能愉快的搭建博客环境

1、安装git

  • 下载地址：https://gitforwindows.org/
  • 安装步骤：双击下载好的.exe文件，一路next就好啦
  • 安装好后，打开git bash输入git version,查看到当前安装的版本号就说明安装成功啦

2、安装node

hexo是基于nodeJS环境的，并且接下来的很多工作都需要用到node中的npm工具

  • node官网：https://nodejs.org/en/
  • 安装步骤：下载好msi文件后，双击打开安装，一路next，不过在Custom Setup这一步记得选 Add to PATH ,这样你就不用自己去配置电脑上环境变量了
  • 控制台输入node -v 出现版本号说明安装成功

3、安装hexo

做完以上两个步骤后我们就可以愉快的安装hexo啦

  • 先创建好一个文件夹（用来存放所有blog的东西），然后打开git bash进入到该文件夹中

  • 安装hexo命令：npm i -g hexo

  • 输入hexo -v，出现版本号说明安装成功

  • 初始化命令：hexo init,初始化完成后你会发现你的文件夹里多了一下的文件

补充说明：

  • node_modules：依赖包
  • public：存放的是生成的页面
  • scaffolds：命令生成文章等的模板
  • source：用命令创建的各种文章
  • themes：主题
  • _config.yml：整个博客的配置
  • db.json：source解析所得到的
  • package.json：项目所需模块项目的配置信息
